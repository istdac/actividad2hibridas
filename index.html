Aplicaciones Híbridas

  • ¿Qué son?

    • § Las aplicaciones híbridas combinan aplicaciones para navegadores web y aplicaciones nativas
    • § Corren como apps reales, sin necesidad de conexión e interfaz con sistema de archivos de un dispositivo
    • § Usualmente hechas en html, css y js envueltos en apps nativas
    • § Aplicaciones web con un contenedor nativo que tiene soporte de hardware (camara, calendario, etc)

  • ¿Cuando usarlas?

    • § Para tomar ventaja de cuando una aplicación web no puede acceder a los dispositivos
    • § Código Reutilizable -> Reducción de costos
    • § Soporte en varias plataformas
    • § Se tiene un equipo familiar con tecnologías web y menos familiar con tecnologías nativas
    • § No se tiene que desarrollar un API desde 0 para que la app funcione

  • Conclusión
    • A pesar de las varias ventajas de las aplicaciones nativas con respecto a su rendimiento, las aplicaciones híbridas, las aplicaciones híbridas tienen las ventajas importantes para poder llegar a ser desarrolladas, ya que las ventajas se ven muy atractivas a las empresas, ya que lo que se puede perder en rendimiento y funcionalidad se puede mitigar con una audiencia mayor y si la aplicación es de calidad, rendir igualmente como una aplicación nativa. Las tecnologías que manejan las aplicaciones híbridas también son populares y ayudan a consolidar el campo de desarrollo.
